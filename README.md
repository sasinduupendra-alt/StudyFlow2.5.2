# StudyFlow Neural

StudyFlow Neural is a high-performance, AI-integrated study management system designed for elite academic performance. It combines strategic objective tracking, temporal schedule synchronization, and neural-based analytics to optimize learning efficiency.

## 🚀 Features

- **Strategic Objectives (Tasks):** Manage your daily, weekly, and monthly goals with a priority-based SNR (Signal-to-Noise Ratio) index.
- **Temporal Matrix (Schedule):** Synchronize your study protocols and tuition alignment in a real-time, interactive schedule.
- **Neural Database (Syllabus):** Track your mastery across multiple subjects with deep-dive topic management.
- **Neural Insights (Analytics):** Leverage AI-processed data to identify weak areas and optimize your focus for maximum retention.
- **Focus Mode:** Immersive study environment with integrated timers and focus tracking.
- **Plan for Tomorrow:** Strategize your next day's objectives in advance to hit the ground running.

## 🛠️ Installation

To set up the project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sasinduupendra/studyflow.git
   cd studyflow
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the root directory and add your Firebase and Gemini API credentials:
   ```env
   VITE_FIREBASE_API_KEY=your_api_key
   VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
   VITE_FIREBASE_PROJECT_ID=your_project_id
   VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
   VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   VITE_FIREBASE_APP_ID=your_app_id
   GEMINI_API_KEY=your_gemini_api_key
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

## 📖 Usage Examples

### Adding a Strategic Objective
1. Navigate to the **Tasks** section.
2. Click **Initialize Objective**.
3. Define your goal, set the impact and effort values, and choose whether to plan for **Today** or **Tomorrow**.
4. Click **Commit Protocol** to save.

### Starting a Focus Session
1. From the **Home** dashboard, select a priority subject or topic.
2. Click the **Play** icon to enter **Focus Mode**.
3. Use the immersive interface to track your session and log your progress upon completion.
